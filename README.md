# Backend Mastering Planner – Sales Page

One-page landing site for the **Backend Mastering Planner** digital product ($5). Built with HTML, CSS, and JavaScript. Dark tech theme, animations, mobile responsive, Formspree form.

## Enable GitHub Pages (free deployment)

1. Open your repo: **https://github.com/fizasaif0233-hash/BackendPlanner**
2. Go to **Settings** → **Pages** (left sidebar).
3. Under **Build and deployment**:
   - **Source:** Deploy from a branch
   - **Branch:** `main` / **Folder:** `/ (root)`
4. Click **Save**.

After a minute or two, the site will be live at:

**https://fizasaif0233-hash.github.io/BackendPlanner/**

## Local run

Open `index.html` in a browser, or run a simple server:

```bash
# Python
python -m http.server 8000

# Node (npx)
npx serve
```

Then visit `http://localhost:8000`.
